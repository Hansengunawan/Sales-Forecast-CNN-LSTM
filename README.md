# Sales-Forecast-CNN-LSTM
PT XYZ faces the problem of uncertain fluctuations in sales of decorative products 
every period, making it difficult for the company to plan stock, organize production, 
and determine effective marketing strategies. Therefore, an accurate and adaptive 
sales prediction system is needed. This study aims to develop a deep learning-based 
sales prediction model using a combination of Convolutional Neural Network 
(CNN) and Long Short-Term Memory (LSTM) and optimize model parameters 
using the Taguchi method with an L16 orthogonal array. The data used is historical 
sales data of PT XYZ's decorative products for the period up to September 2025. 
The research stages include data pre-processing, time series window formation, 
CNN-LSTM model training, hyperparameter optimization using an L16 orthogonal 
array, and model performance evaluation using the Mean Absolute Error (MAE) 
and Root Mean Square Error (RMSE) metrics. The results showed that the best 
configuration of CNN-LSTM with a learning rate of 0.01, batch size 8, epoch 40, 
and RMSprop optimizer produced an MAE value of 1.2281 and an RMSE of 
1.5230. In addition, the results of the comparison with the SARIMA method 
showed that CNN-LSTM had better performance in predicting sales. Thus, the 
CNN-LSTM model optimized using the Taguchi L16 method can be used as an 
effective analytical tool to support PT XYZ's business decision making in data
based inventory, production, and marketing strategy planning.
